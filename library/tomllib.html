tomllib — TOML 파일 구문 분석¶

소스 코드: Lib/tomllib

이 모듈은 TOML 1.1.0 (Tom’s Obvious Minimal Language, https://toml.io\) 구문을 파싱하기 위한 인터페이스를 제공합니다. 이 모듈은 TOML 쓰기를 지원하지 않습니다.

Added in version 3.11: TOML 1.0.0 지원이 추가되었습니다.

버전 3.15에서 변경: TOML 1.1.0 지원이 추가되었습니다. 세부 정보는 What’s New 를 참조하십시오.

경고

신뢰할 수 없는 출처의 데이터를 구문 분석할 때는 주의해야 합니다. 악의적인 TOML 문자열은 디코더가 상당한 CPU 및 메모리 자원을 소모하게 할 수 있습니다. 파싱할 데이터의 크기를 제한하는 것이 좋습니다.

더 보기

Tomli-W 패키지 <tomli-w>`는 이 모듈과 함께 사용될 수 있는 TOML 작성기이며, 표준 라이브러리 :mod:`marshal 및 pickle 모듈을 사용하는 사용자에게 익숙한 쓰기 API를 제공합니다.

더 보기

:pypi:`TOML Kit 패키지 <tomlkit>`은 읽기와 쓰기 기능이 모두 있는 스타일 보존 TOML 라이브러리입니다. 이미 존재하는 TOML 파일을 편집하는 데 추천되는 교체용 모듈입니다.

모듈은 다음 함수들을 정의합니다:

tomllib.load(fp, /, *, parse_float=float)¶

TOML 파일을 읽습니다. 첫 번째 인수는 읽을 수 있고 이진 파일 객체여야 합니다. :class:`dict`를 반환합니다. 이 :ref:`변환 테이블 <toml-to-py-table>`을 사용하여 TOML 타입을 Python으로 변환합니다.

parse_float 는 디코딩할 모든 TOML float의 문자열과 함께 호출됩니다. 기본적으로, 이것은 float(num_str) 와 동등합니다. 이것은 TOML float를 위해 다른 데이터 타입이나 구문 분석기를 사용하는 데 사용될 수 있습니다 (예: decimal.Decimal). 호출 가능한 함수는 dict 나 list 를 반환해서는 안 되며, 그렇지 않으면 ValueError 가 발생합니다.

잘못된 TOML 문서의 경우 :exc:`TOMLDecodeError`가 발생합니다.

tomllib.loads(s, /, *, parse_float=float)¶

Load TOML from a str object. Return a dict. Convert TOML types to Python using this conversion table. The parse_float argument has the same meaning as in load().

잘못된 TOML 문서의 경우 :exc:`TOMLDecodeError`가 발생합니다.

다음과 같은 예외가 사용 가능합니다:

exception tomllib.TOMLDecodeError(msg, doc, pos)¶

다음 추가 속성을 가진 :exc:`ValueError`의 서브클래스입니다:

msg¶

포맷되지 않은 에러 메시지입니다.

doc¶

파싱 중인 TOML 문서입니다.

pos¶

파싱에 실패한 doc 의 인덱스입니다.

lineno¶

pos 에 해당하는 줄입니다.

colno¶

pos 에 해당하는 열입니다.

버전 3.14에서 변경: msg, doc 및 pos 매개변수가 추가되었습니다. msg, doc, pos, lineno 및 colno 속성이 추가되었습니다.

버전 3.14부터 폐지됨: 자유 형식의 위치 인수를 전달하는 것은 더 이상 권장되지 않습니다.

예시:¶

TOML 파일 파싱:

import tomllib

with open("pyproject.toml", "rb") as f:
    data = tomllib.load(f)

TOML 문자열 파싱:

import tomllib

toml_str = """
python-version = "3.11.0"
python-implementation = "CPython"
"""

data = tomllib.loads(toml_str)
