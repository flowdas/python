6. Text Processing Services¶

The modules described in this chapter provide a wide range of string manipulation operations and other text processing services.

The codecs module described under Binary Data Services is also highly relevant to text processing. In addition, see the documentation for Python's built-in string type in 텍스트 시퀀스 형 --- str.
