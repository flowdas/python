암호화 서비스¶

이 장에서 설명하는 모듈은 다양한 암호화 알고리즘을 구현합니다. 가용성은 설치에 달려있습니다. 다음은 개요입니다:

  • hashlib — 보안 해시와 메시지 요약
    • 해시 알고리즘
    • 사용법
    • 생성자
    • 어트리뷰트
    • 해시 객체
    • SHAKE 가변 길이 요약
    • 파일 해싱
    • 키 파생
    • BLAKE2
      • 해시 객체 만들기
      • 상수
      • 예
        • 간단한 해싱
        • 다른 요약 크기 사용하기
        • 키 해싱
        • 무작위 해싱
        • 개인화
        • 트리 모드
      • 크레딧
  • hmac — 메시지 인증을 위한 키 해싱
  • secrets — 비밀 관리를 위한 안전한 난수 생성
    • 난수
    • 토큰 생성
      • 토큰은 몇 바이트를 사용해야 합니까?
    • 기타 함수
    • 조리법과 모범 사례
