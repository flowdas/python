파이썬 2 코드를 파이썬 3으로 이식하는 법¶

Python 2는 2020년 초에 공식적으로 지원이 종료되었습니다. 이는 Python 2에 새로운 버그 보고, 수정 또는 변경이 이루어지지 않음을 의미합니다. 더 이상 지원되지 않으므로 PEP 373 및 `status of Python versions <https://devguide.python.org/versions>`_를 참조하십시오.

순수 파이썬 코드 대신 확장 모듈을 이식하려고 한다면 확장 모듈을 파이썬 3에 이식하기를 참조하십시오.
