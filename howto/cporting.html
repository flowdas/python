확장 모듈을 파이썬 3에 이식하기¶

확장 모듈을 파이썬 3에 이식하는 것에 관해서는 다음과 같은 리소스를 추천합니다:

  • 파이썬 2에서 파이썬 3으로 옮겨가는 것에 관한 일반적인 책인 Supporting Python 3: An in-depth guide의 Migrating C extensions 장은 독자에게 확장 모듈을 이식하는 과정을 안내합니다.

  • py3c 프로젝트의 Porting guide는 지원 코드와 함께 전문적인 제안을 제공합니다.

  • :ref:`권장되는 타사 도구 <c-api-tools>`는 Python의 C API에 대한 추상화를 제공합니다. 확장 기능은 일반적으로 이 중 하나를 사용하도록 재작성되어야 하지만, 라이브러리는 다양한 Python 버전 및 구현 간의 차이점을 처리합니다.
