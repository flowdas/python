PyHash API¶

또한 PyTypeObject.tp_hash 멤버와 :ref:`numeric-hash`를 참조하십시오.

해시 값 유형: 부호 있는 정수입니다.

Added in version 3.2.

type Py_uhash_t¶

해시 값 유형: 부호 없는 정수.

Py_HASH_ALGORITHM¶

str, bytes, 및 :class:`memoryview`의 해싱 알고리즘을 나타내는 숫자 값입니다.

알고리즘 이름은 :data:`sys.hash_info.algorithm`에서 노출됩니다.

Added in version 3.4.

Py_HASH_FNV¶
Py_HASH_SIPHASH24¶
Py_HASH_SIPHASH13¶

해싱에 사용되는 알고리즘을 결정하기 위해 Py_HASH_ALGORITHM 과 비교하는 숫자 값입니다. 해시 알고리즘은 구성 옵션인 --with-hash-algorithm 을 통해 구성할 수 있습니다.

Added in version 3.4: Py_HASH_FNV`와 :c:macro:!Py_HASH_SIPHASH24`를 추가하세요.

Added in version 3.11: :c:macro:`!Py_HASH_SIPHASH13`을 추가하세요.

Py_HASH_CUTOFF¶

Buffers of length in range [1, Py_HASH_CUTOFF) are hashed using DJBX33A instead of the algorithm described by Py_HASH_ALGORITHM.

  • := :c:macro:!Py_HASH_CUTOFF`가 0인 경우 최적화가 비활성화됩니다.

  • :c:macro:`!Py_HASH_CUTOFF`는 음수가 아니어야 하고 7 이하이어야 합니다.

32비트 플랫폼은 충돌하는 문자열을 만들기 쉽기 때문에 64비트 플랫폼보다 더 작은 컷오프를 사용해야 합니다. 64비트 플랫폼에서는 7의 컷오프가, 32비트 플랫폼에서는 5의 컷오프가 적절한 안전 마진을 제공할 것입니다.

이는 sys.hash_info.cutoff 상수를 나타냅니다.

Added in version 3.4.

PyHASH_MODULUS¶

수치 해시 체계에 사용되는 메르센 소수 <https://en.wikipedia.org/wiki/Mersenne_prime>_ P = 2**n -1 입니다.

이는 sys.hash_info.modulus 상수를 나타냅니다.

Added in version 3.13.

PyHASH_BITS¶

PyHASH_MODULUS 에서 P 의 지수 n 입니다.

Added in version 3.13.

PyHASH_MULTIPLIER¶

문자열 및 기타 여러 해시에 사용되는 소수 승수입니다.

Added in version 3.13.

PyHASH_INF¶

양의 무한대에 대해 반환된 해시값입니다.

이는 sys.hash_info.inf 상수를 나타냅니다.

Added in version 3.13.

PyHASH_IMAG¶

복소수의 허수부에 사용되는 승수입니다.

이는 sys.hash_info.imag 상수를 나타냅니다.

Added in version 3.13.

Py_hash_t Py_HashPointer(const void *ptr)¶

포인터 값을 해시합니다: 포인터 값을 정수(내부적으로 uintptr_t 로 캐스팅)로 처리합니다. 포인터는 역참조되지 않습니다.

이 함수는 실패할 수 없습니다: -1 을 반환할 수 없습니다.

Added in version 3.13.

Py_hash_t Py_HashBuffer(const void *ptr, Py_ssize_t len)¶

주소 ptr*에서 시작하는 *len 바이트 크기의 버퍼의 해시값을 계산하여 반환합니다. 해시는 buffer protocol <bufferobjects>`를 구현하는 :class:`bytes, memoryview 및 기타 내장 객체의 해시와 일치하는 것이 보장됩니다.

이 함수를 사용하여 tp_richcompare 함수가 다른 객체의 버퍼와 비교되는 불변 객체의 해싱을 구현합니다.

len 은 0 보다 크거나 같아야 합니다.

이 함수는 항상 성공합니다.

Added in version 3.14.

Py_hash_t PyObject_GenericHash(PyObject *obj)¶

타입 객체의 tp_hash 슬롯에 넣는 것이 의도된 일반 해싱 함수입니다. 그 결과는 객체의 식별자에만 의존합니다.

CPython에서는 :c:func:`Py_HashPointer`와 동등합니다.

Added in version 3.13.
