PyTime C API¶

Added in version 3.13.

클럭 C API는 시스템 시계에 대한 액세스를 제공합니다. 파이썬 time 모듈과 유사합니다.

datetime 모듈의 C API와 관련해서는 :ref:`datetimeobjects`를 참조하십시오.

타입¶

type PyTime_t¶

나노초 단위의 타임스탬프 또는 지속 시간을 나타내며, 부호 있는 64비트 정수입니다.

타임스탬프의 기준점은 사용된 클럭에 따라 달라집니다. 예를 들어, :c:func:`PyTime_Time`은 유닉스 에포크를 기준으로 하는 타임스탬프를 반환합니다.

지원되는 범위는 [-292.3년; +292.3년] 주변입니다. 유닉스 에포크(1970년 1월 1일)를 기준으로, 지원되는 날짜 범위는 [1677-09-21; 2262-04-11] 주변입니다. 정확한 한계는 다음과 같은 상수들로 노출됩니다:

PyTime_t PyTime_MIN¶

:c:type:`PyTime_t`의 최소값입니다.

PyTime_t PyTime_MAX¶

:c:type:`PyTime_t`의 최대값입니다.

클럭 함수¶

다음 함수들은 :c:expr:`PyTime_t`에 포인터를 받아 특정 클럭 값으로 설정합니다. 각 클럭의 세부 정보는 해당 Python 함수의 문서를 참조하십시오.

함수는 성공 시 0 을, 또는 실패 시 -1 (예외 설정됨)을 반환합니다.

정수 오버플로우가 발생하면, PyExc_OverflowError 예외를 설정하고 *result``를 ``[PyTime_MIN; PyTime_MAX] 범위에 제한된 값으로 설정합니다. (현재 시스템에서 정수 오버플로우는 시스템 시간이 잘못 구성된 경우일 가능성이 높습니다.)

다른 C API와 마찬가지로 (별도 지정이 없는 한), 함수를 호출할 때는 :term:`attached thread state`가 필요합니다.

int PyTime_Monotonic(PyTime_t *result)¶

가변 클럭을 읽습니다. 이 클럭에 대한 중요한 세부 정보는 :func:`time.monotonic`을 참조하십시오.

int PyTime_PerfCounter(PyTime_t *result)¶

성능 카운터를 읽습니다. 이 클럭에 대한 중요한 세부 정보는 :func:`time.perf_counter`를 참조하십시오.

int PyTime_Time(PyTime_t *result)¶

“벽시계” 시간을 읽습니다. 이 클럭에 중요한 세부 정보는 :func:`time.time`을 참조하십시오.

원시 클럭 함수¶

클럭 함수와 유사하지만, 에러 발생 시 예외를 설정하지 않으며 호출자에게 :term:`attached thread state`가 필요하지 않습니다.

성공하면 함수는 0 을 반환합니다.

실패한 경우, *result*``를 ``0``으로 설정하고 예외를 설정하지 않은 채 ``-1``을 반환합니다. 오류의 원인을 얻으려면, :term:`attach <attached thread state>`에 :term:`스레드 상태`를 연결하고 일반(비``Raw) 함수를 호출하십시오. 일반 함수는 Raw 함수가 실패한 후에도 성공할 수 있다는 점에 유의하십시오.

int PyTime_MonotonicRaw(PyTime_t *result)¶

:c:func:`PyTime_Monotonic`과 유사하지만, 오류 시 예외를 설정하지 않고 :term:`attached thread state`를 요구하지 않습니다.

int PyTime_PerfCounterRaw(PyTime_t *result)¶

:c:func:`PyTime_PerfCounter`와 유사하지만, 오류 시 예외를 설정하지 않고 :term:`attached thread state`를 요구하지 않습니다.

int PyTime_TimeRaw(PyTime_t *result)¶

:c:func:`PyTime_Time`과 유사하지만, 오류 시 예외를 설정하지 않고 :term:`attached thread state`를 요구하지 않습니다.

변환 함수¶

double PyTime_AsSecondsDouble(PyTime_t t)¶

타임스탬프를 C double 형태의 초 단위 숫자로 변환합니다.

이 함수는 실패할 수 없지만, :c:expr:`double`은 큰 값에 대해 정확도가 제한적이라는 점에 유의하십시오.
