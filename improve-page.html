문서 페이지 개선¶

문서 개선에 대한 아이디어를 언제나 듣고 싶습니다.

“<PAGETITLE>” 페이지를 <PAGEURL>\에서 읽었습니다. 해당 페이지의 소스는 GitHub <https://github.com/python/cpython/blob/main/Doc/PAGESOURCE?plain=1>\에 있습니다.

질문을 하거나 변경을 제안할 수 있는 몇 가지 방법이 있습니다:

  • Python 토론 포럼에서 해당 페이지에 대한 토론을 시작할 수 있습니다. 이 링크를 클릭하면 미리 채워진 토픽이 시작됩니다: Question about page “PAGETITLE” <https://discuss.python.org/new-topic?category=documentation&title=Question+about+page+%22PAGETITLE%22&body=About+the+page+at+PAGEURL%3A>\

  • Python GitHub 이슈 추적기에서 이슈를 열 수 있습니다. 이 링크를 클릭하면 미리 채워진 새 이슈가 생성됩니다: Docs: problem with page “PAGETITLE” <https://github.com/python/cpython/issues/new?template=documentation.yml&title=Docs%3A+problem+with+page+%22PAGETITLE%22&description=The+page+at+PAGEURL+has+a+problem%3A>\

  • 사용하려면 `edit the page on GitHub <https://github.com/python/cpython/blob/main/Doc/PAGESOURCE?plain=1>`_에서 풀 리퀘스트를 열고 기여 절차를 시작할 수 있습니다.
