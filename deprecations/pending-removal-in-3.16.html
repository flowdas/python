Python 3.16에서 제거 예정인 항목들¶

  • 임포트 시스템:

    • 모듈에 16부터는 :attr:!__loader__`가 임포트 시스템이나 표준 라이브러리에서 더 이상 설정되거나 고려되지 않습니다.

    • Setting __package__ on a module while failing to set __spec__.parent is deprecated. In Python 3.16, __package__ will cease to be taken into consideration by the import system or standard library. (gh-97879)

  • The bundled copy of libmpdec.

  • array:

    • The 'u' format code (wchar_t) has been deprecated in documentation since Python 3.3 and at runtime since Python 3.13. Use the 'w' format code (Py_UCS4) for Unicode characters instead.

  • asyncio:

    • asyncio.iscoroutinefunction() is deprecated and will be removed in Python 3.16; use inspect.iscoroutinefunction() instead. (Contributed by Jiahao Li and Kumar Aditya in gh-122875.)

    • asyncio policy system is deprecated and will be removed in Python 3.16. In particular, the following classes and functions are deprecated:

      • asyncio.AbstractEventLoopPolicy

      • asyncio.DefaultEventLoopPolicy

      • asyncio.WindowsSelectorEventLoopPolicy

      • asyncio.WindowsProactorEventLoopPolicy

      • asyncio.get_event_loop_policy()

      • asyncio.set_event_loop_policy()

      Users should use asyncio.run() or asyncio.Runner with loop_factory to use the desired event loop implementation.

      예를 들어, 윈도우에서 asyncio.SelectorEventLoop 클래스를 사용하려면:

      import asyncio

async def main():
    ...

asyncio.run(main(), loop_factory=asyncio.SelectorEventLoop)

      (Kumar Aditya가 gh-127949\를 통해 기능을 구현하였습니다.)

  • :mod:` builtins`:

    • 불리언 타입에 대한 비트 단위 부정(~True 또는 ~False)은 예상치 못한 직관적이지 않은 결과를 산출하기 때문에 Python 3.12부터 사용 지원이 중단되었습니다 (-2 및 -1). 대신 논리적 부정은 not x``를 사용하십시오. 근본적인 정수의 비트 단위 부정이 필요한 경우, 명시적으로 ``int``로 변환하십시오 (``~int(x)).

  • functools:

    • functools.reduce() 의 Python 구현을 function 또는 sequence 를 키워드 인자로 호출하는 방식은 Python 3.14부터 사용 지원이 중단되었습니다.

  • logging:

    • strm 인자를 사용하는 커스텀 로깅 핸들러에 대한 지원은 이제 폐지되었으며 Python 3.16에서 제거될 예정입니다. 대신 stream 인자를 사용하여 핸들러를 정의하십시오. (Mariusz Felisiak가 gh-115032\를 통해 기능을 구현하였습니다.)

  • mimetypes:

    • mimetypes.MimeTypes.add_type`의 유효한 확장자는 '.'로 시작하거나 비어 있어야 합니다. 점으로 시작하지 않는 확장자는 폐지되었으며 Python 3.16에서 :exc:`ValueError()\를 발생시킵니다. (Hugo van Kemenade가 gh-75223\를 통해 기능을 구현하였습니다.)

  • shutil:

    • ExecError 예외는 Python 3.14부터 사용 지원이 중단되었습니다. 이 예외는 Python 3.4 이후 :mod:`!shutil`의 어떤 함수에서도 사용되지 않았으며, 현재는 :exc:`RuntimeError`의 별칭입니다.

  • symtable:

    • symtable.Class.get_methods() 메서드는 Python 3.14부터 사용 지원이 중단되었습니다.

  • sys:

    • _enablelegacywindowsfsencoding() 함수는 Python 3.13부터 사용 지원이 중단되었습니다. 대신 PYTHONLEGACYWINDOWSFSENCODING 환경 변수를 사용하십시오.

  • sysconfig:

    • sysconfig.expand_makefile_vars() 함수는 Python 3.14부터 사용 지원이 중단되었습니다. 대신 sysconfig.get_paths`의 ``vars`() 인자를 사용하십시오.

  • tarfile:

    • 문서화되지 않았고 사용되지 않는 TarInfo.tarfile 속성은 Python 3.13부터 사용 지원이 중단되었습니다.
