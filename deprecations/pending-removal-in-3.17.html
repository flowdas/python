Python 3.17에서 제거 예정인 항목입니다.¶

  • datetime:

    • 년도를 포함하지 않는 형식 문자열 %e (월의 일)를 사용하는 strptime() 호출입니다. 이는 Python 3.15부터 사용 지원 중단(Deprecated)되었습니다. (Stan Ulbrych가 gh-70647 에서 기여).

  • collections.abc:

    • :class:`collections.abc.ByteString`은 Python 3.17에서 제거될 예정입니다.

      런타임에 obj``가 :ref:`버퍼 프로토콜 <bufferobjects>`를 구현했는지 테스트하려면 ``isinstance(obj, collections.abc.Buffer)``를 사용하세요. 타입 주석(type annotation)을 사용할 때는 :class:`~collections.abc.Buffer`를 사용하거나 코드가 지원하는 타입을 명시적으로 지정하는 유니온(union)을 사용하세요 (예: ``bytes | bytearray | memoryview).

      ByteString`은 원래 :class:`bytes`와 :class:`bytearray 모두의 슈퍼타입 역할을 하도록 의도된 추상 클래스였습니다. 하지만 ABC가 메서드를 전혀 가지고 있지 않았기 때문에, 객체가 ByteString`의 인스턴스라는 사실은 객체에 대해 실제로 유용한 정보를 절대 제공하지 못했습니다. :class:`memoryview`와 같은 다른 일반적인 버퍼 타입들도 (실행 시간에 또는 정적 타입 검사기에 의해) :class:!ByteString`의 하위 타입으로 이해되지 않았습니다.

      더 자세한 내용은 :pep:`PEP 688 <688#current-options>`를 참조하십시오. (Shantanu Jain이 :gh:`91896`에서 기여)

  • encodings:

    • :func:`encodings.normalize_encoding`에 비-ASCII encoding 이름을 전달하는 것은 사용이 중단되었으며 Python 3.17에서 제거될 예정입니다. (Stan Ulbrych가 :gh:`136702`에서 기여)

  • webbrowser:

    • webbrowser.MacOSXOSAScript`는 :class:!webbrowser.MacOS`가 선호되면서 사용이 중단되었습니다. (gh-137586)

  • typing:

    • Python 3.14 이전에는 이전 스타일의 유니언이 비공개 클래스 typing._UnionGenericAlias\를 사용해 구현되었습니다. 이 클래스는 구현에 더 이상 필요하지 않지만, 하위 호환성을 위해 유지되어 왔으며 Python 3.17에서 제거될 예정입니다. 사용자는 비공개 구현 세부 사항에 의존하는 대신 typing.get_origin() 및 typing.get_args() 같은 문서화된 인트로스펙션 도우미를 사용해 주시기를 권장합니다.

    • :class:`typing.ByteString`은 Python 3.9부터 사용이 중단되었으며 Python 3.17에서 제거될 예정입니다.

      런타임에 obj``가 :ref:`버퍼 프로토콜 <bufferobjects>`를 구현했는지 테스트하려면 ``isinstance(obj, collections.abc.Buffer)``를 사용하세요. 타입 주석(type annotation)을 사용할 때는 :class:`~collections.abc.Buffer`를 사용하거나 코드가 지원하는 타입을 명시적으로 지정하는 유니온(union)을 사용하세요 (예: ``bytes | bytearray | memoryview).
