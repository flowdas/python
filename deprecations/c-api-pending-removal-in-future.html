미래 버전에서 제거 예정입니다.¶

다음 API들은 더 이상 사용되지 않으며 제거될 예정이지만, 현재 제거 날짜는 예정되어 있지 않습니다.

  • Py_TPFLAGS_HAVE_FINALIZE: Python 3.8부터 불필요합니다.

  • PyErr_Fetch(): 대신 :c:func:`PyErr_GetRaisedException`을 사용하십시오.

  • PyErr_NormalizeException(): 대신 :c:func:`PyErr_GetRaisedException`을 사용하십시오.

  • PyErr_Restore(): 대신 :c:func:`PyErr_SetRaisedException`를 사용하십시오.

  • PyModule_GetFilename(): 대신 :c:func:`PyModule_GetFilenameObject`를 사용하십시오.

  • PyOS_AfterFork(): 대신 :c:func:`PyOS_AfterFork_Child`를 사용하십시오.

  • PySlice_GetIndicesEx(): 대신 PySlice_Unpack() 및 :c:func:`PySlice_AdjustIndices`를 사용하십시오.

  • PyUnicode_READY(): Python 3.12부터 불필요합니다.

  • PyErr_Display(): 대신 :c:func:`PyErr_DisplayException`을 사용하십시오.

  • _PyErr_ChainExceptions(): 대신 :c:func:`!_PyErr_ChainExceptions1`을 사용하십시오.

  • PyBytesObject.ob_shash 멤버: 대신 :c:func:`PyObject_Hash`를 호출하십시오.

  • 스레드 로컬 저장소 (TLS) API:

    • PyThread_create_key(): 대신 :c:func:`PyThread_tss_alloc`를 사용하십시오.

    • PyThread_delete_key(): 대신 :c:func:`PyThread_tss_free`를 사용하십시오.

    • PyThread_set_key_value(): 대신 :c:func:`PyThread_tss_set`를 사용하십시오.

    • PyThread_get_key_value(): 대신 :c:func:`PyThread_tss_get`을 사용하십시오.

    • PyThread_delete_key_value(): 대신 :c:func:`PyThread_tss_delete`를 사용하십시오.

    • PyThread_ReInitTLS(): Python 3.7부터 불필요합니다.
