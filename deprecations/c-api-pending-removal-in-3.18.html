파이썬 3.18에서 제거 예정인 항목들¶

  • 다음 비공개 함수들은 더 이상 사용되지 않으며, Python 3.18에서 제거될 예정입니다:

    • _PyBytes_Join(): :c:func:`PyBytes_Join`을 사용합니다.

    • _PyDict_GetItemStringWithError(): :c:func:`PyDict_GetItemStringRef`를 사용합니다.

    • _PyDict_Pop(): :c:func:`PyDict_Pop`을 사용합니다.

    • _PyLong_Sign(): :c:func:`PyLong_GetSign`을 사용합니다.

    • _PyLong_FromDigits() 및 _PyLong_New(): :c:func:`PyLongWriter_Create`를 사용합니다.

    • _PyThreadState_UncheckedGet(): :c:func:`PyThreadState_GetUnchecked`를 사용하십시오.

    • _PyUnicode_AsString(): :c:func:`PyUnicode_AsUTF8`를 사용하십시오.

    • _PyUnicodeWriter_Init(): _PyUnicodeWriter_Init(&writer) 를 writer = PyUnicodeWriter_Create(0) \를 사용하도록 변경하십시오.

    • _PyUnicodeWriter_Finish(): _PyUnicodeWriter_Finish(&writer) 를 PyUnicodeWriter_Finish(writer) \로 변경하십시오.

    • _PyUnicodeWriter_Dealloc(): _PyUnicodeWriter_Dealloc(&writer) 를 PyUnicodeWriter_Discard(writer) \로 변경하십시오.

    • _PyUnicodeWriter_WriteChar(): _PyUnicodeWriter_WriteChar(&writer, ch) 를 PyUnicodeWriter_WriteChar(writer, ch) \로 변경하십시오.

    • _PyUnicodeWriter_WriteStr(): _PyUnicodeWriter_WriteStr(&writer, str) 를 PyUnicodeWriter_WriteStr(writer, str) \로 변경하십시오.

    • _PyUnicodeWriter_WriteSubstring(): _PyUnicodeWriter_WriteSubstring(&writer, str, start, end) 를 PyUnicodeWriter_WriteSubstring(writer, str, start, end) \로 변경하십시오.

    • _PyUnicodeWriter_WriteASCIIString(): _PyUnicodeWriter_WriteASCIIString(&writer, str) 를 PyUnicodeWriter_WriteASCII(writer, str) \로 변경하십시오.

    • _PyUnicodeWriter_WriteLatin1String(): _PyUnicodeWriter_WriteLatin1String(&writer, str) 를 PyUnicodeWriter_WriteUTF8(writer, str) \로 변경하십시오.

    • _PyUnicodeWriter_Prepare(): (다른 직접적인 대안이 없습니다.)

    • _PyUnicodeWriter_PrepareKind(): (다른 직접적인 대안이 없습니다.)

    • _Py_HashPointer(): :c:func:`Py_HashPointer`를 사용하십시오.

    • _Py_fopen_obj(): :c:func:`Py_fopen`을 사용하십시오.

    • PyGen_New(): (대체 없음).

    • PyGen_NewWithQualName(): (대체 없음).

    • PyCoro_New(): (대체 없음).

    • PyAsyncGen_New(): (대체 없음).

    pythoncapi-compat project <https://github.com/python/pythoncapi-compat/>`__는 Python 3.13 및 이전 버전에서 이러한 새로운 공개 함수들을 얻는 데 사용될 수 있습니다. (Victor Stinner가 :gh:`128863\를 통해 기여했습니다.)
