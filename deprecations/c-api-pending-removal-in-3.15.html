Python 3.15에서 제거 예정¶

  • PyImport_ImportModuleNoBlock(): 대신 :c:func:`PyImport_ImportModule`을 사용해주시기를 부탁드립니다.

  • PyWeakref_GetObject`와 :c:func:()!PyWeakref_GET_OBJECT`: 대신 :c:func:`PyWeakref_GetRef`를 사용하세요. Python 3.12 및 이전 버전에서 :c:func:`PyWeakref_GetRef`를 얻으려면 `pythoncapi-compat 프로젝트 <https://github.com/python/pythoncapi-compat/>`__을 사용할 수 있습니다.

  • PyUnicode_AsDecodedObject(): 대신 :c:func:`PyCodec_Decode`를 사용하세요.

  • PyUnicode_AsDecodedUnicode(): 대신 :c:func:`PyCodec_Decode`를 사용하세요. 일부 코덱(예를 들어, “base64”)은 :class:`str`이 아닌 :class:`bytes`와 같은 다른 타입을 반환할 수 있음에 유의하세요.

  • PyUnicode_AsEncodedObject(): 대신 :c:func:`PyCodec_Encode`를 사용하세요.

  • PyUnicode_AsEncodedUnicode(): 대신 :c:func:`PyCodec_Encode`를 사용하세요. 일부 코덱(예를 들어, “base64”)은 :class:`bytes`가 아닌 :class:`str`와 같은 다른 타입을 반환할 수 있음에 유의하세요.

  • Python 초기화 함수들, Python 3.13에서 사용 지원이 중단(deprecated)되었습니다:

    • Py_GetPath(): 대신 PyConfig_Get("module_search_paths") (sys.path)를 사용하세요.

    • Py_GetPrefix(): Use PyConfig_Get("base_prefix") (sys.base_prefix) instead. Use PyConfig_Get("prefix") (sys.prefix) if virtual environments need to be handled.

    • Py_GetExecPrefix(): Use PyConfig_Get("base_exec_prefix") (sys.base_exec_prefix) instead. Use PyConfig_Get("exec_prefix") (sys.exec_prefix) if virtual environments need to be handled.

    • Py_GetProgramFullPath(): Use PyConfig_Get("executable") (sys.executable) instead.

    • Py_GetProgramName(): Use PyConfig_Get("executable") (sys.executable) instead.

    • Py_GetPythonHome(): Use PyConfig_Get("home") or the PYTHONHOME environment variable instead.

    The pythoncapi-compat project can be used to get PyConfig_Get() on Python 3.13 and older.
