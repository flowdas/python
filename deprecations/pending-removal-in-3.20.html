Python 3.20에서 제거 예정인 항목들¶

  • struct.Struct`의 ``__new__()` 메서드를 format 인자 없이 호출하는 것은 더 이상 사용되지 않으며 Python 3.20에서 제거될 예정입니다. 초기화된 Struct 객체에 __init__() 메서드를 호출하는 것도 더 이상 사용되지 않으며 Python 3.20에서 제거될 예정입니다.

    (Sergey B Kirpichev가 gh-143715\를 통해 기능을 구현하였습니다. Serhiy Storchaka가 gh-143715\를 통해 기능을 구현하였습니다.)

  • __version__, version, 및 VERSION 속성은 다음 표준 라이브러리 모듈에서 더 이상 사용되지 않으며 Python 3.20에서 제거될 예정입니다. 대신 :py:data:`sys.version_info`를 사용하십시오.

    • argparse

    • csv

    • ctypes

    • ctypes.macholib

    • decimal (대신 :data:`decimal.SPEC_VERSION`을 사용하십시오)

    • http.server

    • imaplib

    • ipaddress

    • json

    • logging (`__date__`도 더 이상 사용하지 않습니다)

    • optparse

    • pickle

    • platform

    • re

    • socketserver

    • tabnanny

    • tarfile

    • tkinter.font

    • tkinter.ttk

    • wsgiref.simple_server

    • xml.etree.ElementTree

    • xml.sax.expatreader

    • xml.sax.handler

    • zlib

    (Hugo van Kemenade와 Stan Ulbrych가 gh-76007\를 통해 기능을 구현하였습니다.)

  • :pep:`829`에 의해 정의된 폐지 사항:

    • name.pth 파일에서 발견된 import 줄에 대해 경고가 발생합니다.

    • name.pth 파일은 기본적으로 로케일 인코딩으로 더 이상 디코딩되지 않습니다. 반드시 ``utf-8-sig``로 인코딩되어야 합니다.

    (Barry Warsaw가 gh-148641\를 통해 기능을 구현하였습니다.)

  • ast:

    • 추상 AST 노드 (예: ast.AST 또는 ast.expr)의 인스턴스를 생성하는 것은 더 이상 권장되지 않으며 Python 3.20에서 오류를 발생시킬 것입니다.
