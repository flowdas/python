Python 3.20에서 제거 예정입니다.¶

  • _PyObject_CallMethodId(), _PyObject_GetAttrId() 및 _PyUnicode_FromId() 는 3.15 버전에서 사용이 중단되며 3.20 버전에서 제거될 예정입니다. 대신 PyUnicode_InternFromString() 을 사용하고 그 결과를 모듈 상태에 캐시한 다음, PyObject_CallMethod() 또는 PyObject_GetAttr() 를 호출하십시오. (Victor Stinner가 gh-141049 \를 통해 기능을 구현하였습니다.)

  • PyComplexObject (gh-128813)에서 cval 필드를 사용합니다. Py_complex 표현식으로 파이썬 복소수를 변환하려면 :c:func:`PyComplex_AsCComplex`와 :c:func:`PyComplex_FromCComplex`를 사용하십시오.

  • 매크로 Py_MATH_PIl 및 Py_MATH_El.
