Python 3.15에서 제거 예정¶

  • 임포트 시스템:

    • 모듈에 __cached__ 를 설정하는 동시에 __spec__.cached \를 설정하지 않는 방식은 사용 지원이 중단되었습니다. Python 3.15부터는 __cached__ 가 임포트 시스템이나 표준 라이브러리에서 설정되거나 고려되지 않을 것입니다. (gh-97879 를 참고하십시오.)

  • ctypes:

    • 문서화되지 않은 ctypes.SetPointerType() 함수는 Python 3.13부터 사용 지원이 중단되었습니다.

  • http.server:

    • 구식이 되었고 거의 사용되지 않는 :class:`!CGIHTTPRequestHandler`는 Python 3.13부터 사용 지원이 중단되었습니다. 직접적인 대체 수단은 없습니다. 어떤 것이 요청 핸들러와 웹 서버를 연결하는 데 CGI보다 낫다는 점에 유의하십시오.

    • python -m http.server 명령줄 인터페이스의 --cgi 플래그는 Python 3.13부터 사용 지원이 중단되었습니다.

  • importlib:

    • load_module() 메서드: 대신 exec_module() 를 사용하십시오.

  • pathlib:

    • :meth:`!.PurePath.is_reserved`는 Python 3.13부터 사용 지원이 중단되었습니다. 윈도우에서 예약된 경로를 감지하려면 :func:`os.path.isreserved`를 사용하십시오.

  • platform:

    • platform.java_ver() 함수는 Python 3.13부터 사용 지원이 중단되었습니다. 이 함수는 Jython 지원에만 유용하며, API가 혼란스럽고, 대부분의 경우 테스트되지 않았습니다.

  • sysconfig:

    • :func:`sysconfig.is_python_build`의 check_home 인자는 Python 3.12부터 사용 지원이 중단되었습니다.

  • threading:

    • :func:`~threading.RLock`는 Python 3.15부터 인자를 받지 않습니다. 인자를 전달하는 것은 Python 3.14부터 사용 지원이 중단되었습니다. 파이썬 버전은 어떤 인자도 허용하지 않지만, C 버전은 위치 인자나 키워드 인자를 임의의 개수만큼 허용하며 모든 인자를 무시하기 때문입니다.

  • types:

    • types.CodeType: Accessing codeobject.co_lnotab was deprecated in PEP 626 since 3.10 and was planned to be removed in 3.12, but it only got a proper DeprecationWarning in 3.12. May be removed in 3.15. (Contributed by Nikita Sobolev in gh-101866.)

  • typing:

    • NamedTuple 클래스를 생성하기 위한 문서화되지 않은 키워드 인자 구문은 Python 3.13부터 사용을 중단하였습니다. 대신 클래스 기반 구문이나 함수형 구문을 사용해 주십시오.

    • TypedDict\을 함수형 구문으로 사용할 때 fields 매개변수에 값을 전달하지 않거나 (TD = TypedDict("TD")) None\을 전달하는 것 (TD = TypedDict("TD", None))은 Python 3.13부터 사용을 중단했습니다. 0개의 필드를 가진 TypedDict를 생성하려면 class TD(TypedDict): pass\이나 TD = TypedDict("TD", {})\를 사용하십시오.

    • @typing.no_type_check_decorator 데코레이터 함수는 Python 3.13부터 사용을 중단하였습니다. typing 모듈에 추가된 지 8년이 지났지만, 아직까지 어떤 주요 타입 검사기에서도 이를 지원하지 않고 있습니다.

  • sre_compile, sre_constants 및 sre_parse 모듈.

  • wave:

    • Wave_write 클래스의 getmark(), setmark() 및 getmarkers() 메서드는 Python 3.13부터 사용을 중단하였습니다.

  • zipimport:

    • zipimport.zipimporter.load_module() 메서드는 Python 3.10부터 사용을 중단하였습니다. 대신 exec_module()\를 사용해 주십시오. (gh-125746.)
