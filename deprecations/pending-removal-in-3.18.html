Python 3.18에서 제거 예정입니다.¶

  • 더 이상 파일 디스크립터가 예상되는 곳에 불리언 값을 허용하지 않습니다. (Serhiy Storchaka가 :gh:`82626`\를 통해 기능을 구현하였습니다.)

  • decimal:

    • 비표준이며 문서화되지 않은 Decimal 형식 지정자 ‘N’`는 Python 3.13부터 폐지되었으며, 이 지정자는 :mod:!decimal` 모듈의 C 구현에서만 지원됩니다. (Serhiy Storchaka가 :gh:`89902`\를 통해 기능을 구현하였습니다.)

  • :pep:`829`에 의해 정의된 지원 중단 항목:

    • name.pth 파일의 import 줄은 조용히 무시됩니다.

    (Barry Warsaw가 :gh:`148641`\를 통해 기능을 구현하였습니다.)
