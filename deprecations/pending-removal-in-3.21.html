Python 3.21에서 제거 예정인 항목¶

  • abc

    • Python 3.3부터 소프트-폐지된 abc.abstractclassmethod, abc.abstractstaticmethod, 그리고 abc.abstractproperty\는 이제 DeprecationWarning`를 발생시킵니다. 이 클래스들은 Python 3.21에서 제거될 예정이므로, 대신 각각 :func:`abc.abstractmethod\와 함께 classmethod(), staticmethod(), 그리고 :class:`property`를 사용하십시오.

  • ast:

    • 클래스 slice, Index, ExtSlice, Suite, Param, AugLoad, 그리고 AugStore\는 Python 3.21에서 제거됩니다. 이 타입들은 파서에 의해 생성되지 않았거나 코드 생성기에 의해 받아들여지지 않았습니다.

    • ast.Tuple\의 dims 속성은 Python 3.21에서 제거됩니다. 대신 ast.Tuple.elts 속성을 사용하십시오.
