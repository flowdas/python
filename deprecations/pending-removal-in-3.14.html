Python 3.14에서 제거 예정인 항목들¶

  • argparse: :class:`!argparse.BooleanOptionalAction`의 type, choices, 및 metavar 매개변수는 사용 중단되었으며 3.14에서 제거될 예정입니다. (Nikita Sobolev가 :gh:`92248`을 통해 기능을 구현하였습니다.)

  • ast: 다음 기능들은 Python 3.8 이후 문서에서 사용 중단되었으며, 이제 접근하거나 사용될 때 런타임에 :exc:`DeprecationWarning`가 발생하고 Python 3.14에서 제거될 예정입니다:

    • ast.Num

    • ast.Str

    • ast.Bytes

    • ast.NameConstant

    • ast.Ellipsis

    :class:`ast.Constant`를 사용하십시오. (Serhiy Storchaka가 :gh:’90953’를 통해 기능을 구현하였습니다.)

  • asyncio:

    • 자식 관찰자 클래스(child watcher class)인 asyncio.MultiLoopChildWatcher, asyncio.FastChildWatcher, asyncio.AbstractChildWatcher, 그리고 :class:`!asyncio.SafeChildWatcher`는 사용 중단되었으며 Python 3.14에서 제거될 예정입니다. (Kumar Aditya가 :gh:`94597`을 통해 기능을 구현하였습니다.)

    • asyncio.set_child_watcher(), asyncio.get_child_watcher(), asyncio.AbstractEventLoopPolicy.set_child_watcher() 및 :meth:`!asyncio.AbstractEventLoopPolicy.get_child_watcher`는 사용 중단되었으며 Python 3.14에서 제거될 예정입니다. (Kumar Aditya가 :gh:`94597`을 통해 기능을 구현하였습니다.)

    • 기본 이벤트 루프 정책의 get_event_loop() 메서드는 현재 이벤트 루프가 설정되어 있지 않아 생성하기로 결정하는 경우 DeprecationWarning`을 발생시킵니다. (Serhiy Storchaka와 Guido van Rossum이 :gh:`100160\를 통해 기능을 구현하였습니다.)

  • email: 함수 email.utils.localtime`에 포함된 *isdst* 매개변수를 폐지했습니다. (Alan Williams가 :gh:`72346()\를 통해 기능을 구현하였습니다.)

  • :mod:`importlib.abc`의 폐지된 클래스들:

    • importlib.abc.ResourceReader

    • importlib.abc.Traversable

    • importlib.abc.TraversableResources

    대신 importlib.resources.abc 클래스를 사용하십시오:

    • importlib.resources.abc.Traversable

    • importlib.resources.abc.TraversableResources

    (Jason R. Coombs와 Hugo van Kemenade가 gh-93963\를 통해 기능을 구현하였습니다.)

  • itertools`는 copy, deepcopy, 및 pickle 연산에 대해 문서화되지 않은, 비효율적이고, 역사적으로 버그가 많으며, 일관성 없는 지원을 했습니다. 코드를 대폭 줄이고 유지보수 부담을 줄이기 위해 3.14 버전에서 이 지원이 제거될 것입니다. (Raymond Hettinger가 :gh:`101588\를 통해 기능을 구현하였습니다.)

  • multiprocessing: 기본 시작 방식이 리눅스, BSD 및 기타 비-macOS POSIX 플랫폼에서 더 안전한 방식으로 변경됩니다. 현재 이 플랫폼들에서는 'fork' 가 기본값입니다 (gh-84559 \). 대부분의 코드는 신경 쓰지 않을 것으로 예상되기 때문에, 이 변경에 대한 런타임 경고를 추가하는 것은 너무 방해가 될 수 있습니다. 코드가 'fork' 가 필수인 경우 get_context() \ 또는 set_start_method() \ API를 사용하여 명시적으로 지정하십시오. 컨텍스트 및 시작 방법 를 참조하십시오.

  • pathlib: is_relative_to()\와 relative_to():\에 추가 인자를 전달하는 것은 폐지되었습니다.

  • pkgutil: pkgutil.find_loader`와 :func:()!pkgutil.get_loader`가 이제 DeprecationWarning`을 발생시키므로, 대신 :func:`importlib.util.find_spec`을 사용하십시오. (Nikita Sobolev가 :gh:`97850\를 통해 기능을 구현하였습니다.)

  • pty:

    • master_open(): pty.openpty()\를 사용해주십시오.

    • slave_open(): pty.openpty()\를 사용해주십시오.

  • sqlite3:

    • version`과 :data:!version_info`입니다.

    • named placeholders 를 사용하고 parameters 가 dict 가 아닌 시퀀스인 경우, execute() 와 executemany() 를 사용하십시오.

  • urllib: urllib.parse.Quoter`는 폐지되었습니다: 공개 API가 의도된 것이 아닙니다. (Gregory P. Smith가 :gh:`88168\를 통해 기능을 구현하였습니다.)
